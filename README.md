# Customer Shopping Dashboard – Power BI

I built this dashboard to practice analyzing customer behavior data. Nothing fancy, just wanted to get comfortable with Power BI and see what insights I could pull from a shopping dataset.

## What you're looking at

The dashboard tracks a few main things:

- **1,844** previous purchases (total across all customers)
- **4,326** total amount spent
- **65** items purchased
- **3.69** average customer rating (out of 5)
- **45.28** average customer age

Nothing groundbreaking, but it was a good exercise in cleaning data and setting up KPIs.

## Files in this repo

- `Customer Shopping.pdf` – exported dashboard, easiest to view
- `Customer Shopping.pbix` – the actual Power BI file if you want to poke around
- `customer_shopping_behavior.csv` – raw data I used

## A few things I noticed

**Customer age** – average is around 45. That's useful to know if you're targeting ads or planning promotions.

**Ratings** – 3.69 is okay but not great. Probably means some customers are happy, some aren't. Would be worth digging into the low ratings.

**Purchase volume** – 65 items across 1,844 purchases suggests people buy multiple items per trip. Cross-selling might work here.

## How to view it

If you just want to see what it looks like – open the PDF. 

If you have Power BI Desktop (free download), you can open the .pbix file and click around. The CSV is there if you want to see the original data.
